# Operational Analytics Dashboard | SQL + Power BI

##  Visão Geral
Projeto de análise de dados operacionais focado em **performance, qualidade de dados e automação de processos**, utilizando **SQL e Power BI**.

O objetivo é demonstrar como dados operacionais podem ser transformados em **indicadores confiáveis** para apoio à tomada de decisão.

---

##  Problema de Negócio
Processos operacionais geram grande volume de dados, porém sem padronização e indicadores claros de:
- Eficiência operacional
- Tempo de processamento
- Qualidade e consistência dos dados
- Ocorrência de erros

Sem essas informações, a tomada de decisão fica reativa.

---

##  Objetivo
- Estruturar dados operacionais em um **modelo analítico (Star Schema)**
- Garantir **qualidade e governança dos dados**
- Criar **KPIs e dashboards** para acompanhamento da performance

---

##  Modelagem de Dados
- **Star Schema**
- Tabela fato de processamento
- Dimensões de data, processo, sistema, status, moeda e tipo de erro
- Tabela específica para validação de qualidade de dados

---

##  Tecnologias Utilizadas
- SQL (modelagem, views, agregações)
- Power BI (visualizações e KPIs)
- Excel / CSV (dados simulados)
- Conceitos de ETL e governança de dados

---

##  Indicadores Criados
- Total de execuções
- Tempo médio de processamento
- Percentual de erro
- Throughput diário
- Qualidade de dados por campo

---

##  Resultados
- Visão clara de gargalos operacionais
- Identificação de processos com maior incidência de erros
- Base confiável para análises futuras e automações
